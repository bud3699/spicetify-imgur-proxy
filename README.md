# imgur-proxy

Spicetify extension to proxy Imgur images through DuckDuckGo for better accessibility in the UK.

## Installation

1. Download `imgur-proxy.js`
2. Move it to your Spicetify extensions folder
3. Run:
   ```bash
   spicetify config extensions imgur-proxy.js
   spicetify apply
    ```
